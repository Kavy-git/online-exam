# online-exam
A secure online examination platform with real-time monitoring, test scheduling, and automated evaluation using the MERN stack.

---

## 🚀 Features

- Secure user authentication
- Create and attempt tests
- Test scheduling with time validation
- Real-time monitoring (anti-cheating)
- Automatic result analysis

---

## 🛠️ Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB Atlas

---

## 📦 Installation

### Backend
cd backend  
npm install  
npm run server  

### Frontend
cd frontend  
npm install  
npm run dev  

---

## 🔑 Environment Variables

MONGO_URI=your_mongodb_url  
JWT_SECRET=your_secret_key  
PORT=4000  

---

## 🌍 Usage

Frontend: http://localhost:5173  
Backend: ${import.meta.env.VITE_API_URL}
